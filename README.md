# 豆瓣爬虫
## 使用方法：
> 运行main.py即可进入主界面

> 功能：抓取豆瓣电影各分类评分前N部，并分析电影所属区域、电影时间长度、电影最多评论、最高评分、电影年代分布、电影类型分布等信息
